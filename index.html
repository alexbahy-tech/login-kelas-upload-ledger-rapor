üìä TAHAP 2: MENGGUNAKAN DASHBOARD E-RAPOR

1. Memilih Mode Kerja

Aplikasi memiliki 2 mode yang bisa dipilih:

  • Mode Rapor: Untuk mengelola data rapor siswa
  • Mode Identitas Siswa: Untuk mengelola data identitas siswa

2. Mengatur Filter Data

Pilih parameter berikut untuk menampilkan data yang diinginkan:

  • Tahun Pelajaran: Pilih antara TP 2025/2026, TP 2026/2027, atau TP 2027/2028
  • Semester: Pilih Ganjil atau Genap
  • Klik tombol "Cek Status" untuk memuat data

3. Memeriksa Status Ledger

Setelah mengklik "Cek Status", sistem akan menampilkan:

  • Kelengkapan Rapor: Jumlah siswa yang sudah lengkap rapornya dan persentase
  • Kelengkapan Identitas: Jumlah siswa yang sudah lengkap identitasnya dan persentase

‚ûï TAHAP 3: MENGELOLA DATA SISWA

Menambah Data Siswa

  • Klik tombol "Tambah Siswa" di bagian bawah

  • Isi data siswa yang diperlukan
  • Klik "SIMPAN" untuk menyimpan atau "BATAL" untuk membatalkan
